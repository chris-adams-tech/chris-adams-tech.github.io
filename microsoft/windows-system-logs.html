Windows System Logs

Event ID 1074

System Shutdown/Restart - indicates when and why the system was shut down or restarted

  • can indicate if there was an unexpected shutdown or restart
  • could potentially reveal malicious activity
    • malware infection
    • unauthorized access

Event ID 6005

The Event log service was started - marks time when Event Log Service starts

  • indicates system boot-up
  • provides starting point for investigating system performance or potential surrounding security incidents
  • detects unauthorized system reboots

Event ID 6006

The Event log service was stopped - marks time when Event Log Service stops

  • indicates system shut down
  • abnormal or unexpected occurrences could point to intentional service disruption or covering tracks

Event ID 6013

Windows uptime - occurs once a day and show uptime of system in seconds

  • shorter than expected could mean system has been rebooted
    • an indication of potential intrusion or unauthorized activity

Event ID 7040

Service status change - change in service startup type

  • could be from manual to automatic or vice versa
  • could indicate sign of tampering

Windows Security Logs

Event ID 1102

The audit log was cleared

  • can be an attempt to remove evidence

Event ID 1116

Antivirus malware detection

  • logs when Defender detects malware

Event ID 1118

Antivirus remediation activity has started

  • indicates when Defender has begun process of removing or quarantining detected malware

Event ID 1119

Antivirus remediation activity has succeeded

  • indicates remediation for detected malware has been successful

Event ID 1120

Antivirus remediation activity has failed

  • indication remediation has failed
  • should be closely monitored

Event ID 4624

Successful Logon

  • records successful logons
  • be cautious of abnormal behavior, such as logons at odd hours or different locations

Event ID 4625

Failed Logon

  • records failed logons
  • multiple attempts could signify brute force attack

Event ID 4648

A logon was attempted using explicit credentials

  • user logons with explicit credentials to run a program
  • anomalies could indicate lateral movement

Event ID 4656

A handle to an object was requested

  • requesting access to object such as file, registry key, or process
  • detects attempts to view sensitive resources

Event ID 4672

Special Privileges Assigned to a New Logon

  • logons for users with super user privileges
  • ensure to always implement least privilege

Event ID 4698

A scheduled task was created

  • can help detect persistence techniques
  • attackers often use scheduled tasks to maintain access and run malicious code

Event ID 4700 & Event ID 4701

A scheduled task was enabled/disabled

  • enabling and disabling of a task
  • can provide valuable insights if schedules tasks are manipulated by attackers

Event ID 4702

A scheduled task was updated

  • can help detect unauthorized changes to scheduled tasks

Event ID 4719

System audit policy was changed

  • attacker could cover tracks by turning off auditing or modifying auditing

Event ID 4738

A user account was changed

  • any changes made to user accounts
    • privileges, group memberships, account settings
  • can be a sign of account takeover or insider threats

Event ID 4771

Kerberos pre-authentication failed

  • unusual amount could indicate brute force on Kerberos service

Event ID 4776

The domain controller attempted to validate the credentials for an account

  • tracks both successful and unsuccessful failed attempts at credential validation by a DC
  • multiple failures could suggest brute force attack*

Event ID 5001

Antivirus real-time protection configuration has changed

  • unauthorized changes could indicate an attempt to disable functionality of Defender

Event ID 5140

A network share object was accessed

  • identifies unauthorized access to network shares

Event ID 5142

A network share object was added

  • creation of new network share
  • unauthorized network shares could be used to exfiltrate data or spread malware

Event ID 5145

A network share object was checked to see whether client can be granted desired access

  • someone attempted to access a network share
  • frequent checks might indicate a user or malware trying to map out network shares for future exploits

Event ID 5157

The Windows Filtering Platform has blocked a connection

  • helpful for identifying malicious traffic on network

Event ID 7045

A service was installed in the system

  • sudden appearance of unknown service might suggest malware installation*
