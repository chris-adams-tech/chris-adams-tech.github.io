Navigating the Canvas Mindmap pages

At this point, if you press tab, a new child card will be created.

To move a card using hotkeys, hold SHIFT, then use arrow keys to move the card

You can also create a sibling card by pressing, CTRL + ENTER, while in Viewing mode.

Multiple cards...

Can be created by holding SHIFT + ENTER for each card

Once a card is created, exit the edit mode by clicking ESC

To start the workflow, create a card.

To move around to different cards, hold ALT, then use arrow keys

Create a new unconnected card by holding CTRL then pressing the Down Arrow

For scrolling:

  • Vertical: mouse wheel
  • Horizontal: Hold SHIFT, then use the mouse wheel.
  • Zoom: Hold Windows/Super key, then use mouse wheel.
