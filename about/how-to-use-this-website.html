How to use this website

When first landing here, it can be a bit overwhelming with all the things and stuff everywhere, you might think, what does this guy actually do? I'm not really sure of it yet either.

You'll notice throughout this website there are borders around some images. I tend to mostly use bright green and red, however you will notice a more turquoise or teal color. This is mostly to create a border for screenshots or images. I tent to use dark mode on everything and help to not blend in with everything.

Here are some navigation tips:

The Interactive Graph

You'll notice on the right hand side there is this mind map graph looking thing. I really like this feature as it's an amazing way to visualize the connections between your pages.

The above graph is showing a representation of all the files and folders that are listed in the Table of Contents.

  1. This arrow button expands to full screen to view the interactive graph.
  2. Zoom by scrolling in/out. Click on any dot to bring you to the document.
  3. Shows the Title of the page under the Table of Contents then the following headings and subheadings that follow in that page.
    1. Note: Everything listed in this table will be only on that page. There could be links that go to other pages.

So how do the pages become linked?

Anytime you create a backlink within a page using syntax [[page-name]], where page-name is the name of the page that you would like to link to.

Here's an example. Let's create a test page. Test Page Once we create the test page, notice that it's populated with a connection in the local graph. Since we are on the How to use this website page, on the graph the circle is illuminated purple indicating that is the file we are viewing.

Once we click on the Test Page, notice that the circle that was illuminated purple before is now grayed out and the Test Page is purple, showing us we are in the connected page.
