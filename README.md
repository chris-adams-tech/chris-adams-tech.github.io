# About this repository

This repo is hosting my vault from Obsidian via GitHub pages with a Jekyll workflow for deployment. I use this plugin to export the `md` files to `html` code then Jekyll pushes the deploy to chris-adams-tech.github.io

Here is a link to the GitHub repo for the HTML export: https://github.com/KosmosisDire/obsidian-webpage-export

Below is a screenshot from the website from this repository:
![image](https://github.com/user-attachments/assets/d91159e0-af95-4bd0-8562-318e94a29a1f)

There may be recurring updates to new and existing projects. I am a true believer in continuous learning and am never fully complete with a project ;)

My goal is to create a repository of information that can be searched in various different ways and am still learning how to best do that. This project serves as an anchor for me to create a system of understanding complex topics in a non-linear fashion.

The content within all of the pages are a culmination of my studies from school as well as post graduate studies, mostly into cyber security, but some into development and automation. 

Check in here for more updates! I hope to continue to integrate new plugins and features.
